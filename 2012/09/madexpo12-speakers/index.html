MADExpo 2012 Speaker Blogs and Twitter Accounts

2012-09-29

Here are the MADExpo 2012 speaker blogs and Twitter accounts. Enjoy!

BLOG FEED FILE

You can subscribe to the MADExpo 2012 speaker blogs by importing this OPML file into your feed reader.

If you aren’t familiar with OPML, it is an xml file that contains feed urls. You can easily import an OPML file into your feed reader and subscribe to all the feeds in bulk. If you don’t know how to do this then just search for “OPML import [feed reader name]” (replace [feed reader name] with the feed reader you use). Ping me if you have any trouble and I’ll do my best to help you out.

http://jongallant.com/MADExpo12SpeakerBlogs.xml

# TWITTER LIST

You can easily follow all the MADExpo 2012 speakers by going to the MADExpo 2012 Speaker list page and clicking “Follow” next to the speakers that you want to follow.
