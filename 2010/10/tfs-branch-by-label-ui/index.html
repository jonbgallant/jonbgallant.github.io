TFS 2010: "Branch by Label" is back in the UI

2010-10-26

TFS 2010 was released without “Branch by Label” in the UI (you had to do using TF.exe /branch /version:Lxxx).

A hotfix was released on Aug 3 that fixes that problem.

download: http://code.msdn.microsoft.com/KB983504/Release/ProjectReleases.aspx?ReleaseId=4749
